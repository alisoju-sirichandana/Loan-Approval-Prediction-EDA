# Loan-Approval-Prediction-EDA
Exploratory Data Analysis on loan approval dataset using Python and Streamlit.

## Objective
To analyze loan application data and identify key factors influencing loan approval decisions.

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook
- Streamlit

## Project Workflow
1. Data loading and inspection
2. Data cleaning and preprocessing
3. Exploratory Data Analysis (EDA)
4. Visualization of key insights
5. Interactive EDA dashboard using Streamlit

## Key Insights
- Credit history has strong impact on loan approval
- Applicant income and loan amount influence approval chances
- Married applicants show higher approval rates
